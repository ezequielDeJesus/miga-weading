# Miga 💍✨ — Assessoria de Casamento Inteligente

**Miga** é uma assistente virtual de luxo e uma suite de ferramentas criativas projetada para transformar o planejamento de casamentos em uma experiência editorial, intuitiva e mágica. Combinando tecnologia de ponta e um design premium, Miga ajuda noivas a visualizarem seus sonhos antes mesmo deles acontecerem.

## 🔥 Funcionalidades Principais

### 👗 Provador Mágico (AI Try-On)
Experimente vestidos de noiva de forma realista usando a **Nano Banana AI**. Através de uma composição avançada em HTML5 Canvas, o sistema funde a foto da noiva com o vestido escolhido, aplicando iluminação de estúdio e efeitos editoriais.

### 🖼️ Decorador de Sonhos
Visualize a ambientação do seu grande dia. Faça o upload da foto do local e escolha estilos como Clássico, Luxuoso ou Minimalista. A IA projeta elementos decorativos para dar vida ao cenário.

### 💬 Miga Chat
Uma IA companheira que entende as emoções da jornada da noiva. Miga oferece suporte, tira dúvidas e ajuda na curadoria de inspirações com uma interface fluida e acolhedora.

### 📓 Moodboard Interativo
Um scrapbook digital para organizar referências. Cada inspiração é tratada como uma fotografia polaroid, permitindo uma visualização tátil e charmosa do conceito visual do casamento.

## 🎨 Design System

O projeto segue uma estética **Editorial Boutique**:
- **Paleta de Cores**: Dusty Rose (`#b5827e`), Azul Profundo (`#225373`) e Fendi (`#fdfbf7`).
- **Tipografia**: Uma curadoria de fontes que mistura o moderno (*Plus Jakarta Sans*) com o toque pessoal do manuscrito (*Dancing Script*).
- **UX**: Navegação fluida, micro-animações e feedback visual imersivo.

## 🚀 Como Executar

Este projeto foi construído utilizando **Vite** e **Tailwind CSS v4**.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/ezequielDeJesus/miga-sua-miga.git
    cd miga-sua-miga
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Acesse no navegador:**
    Normalmente em `http://localhost:5173`.
